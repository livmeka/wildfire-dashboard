# US Wildfire Dashboard



An interactive Panel dashboard exploring US wildfire patterns from 1992–2020.



🔗 **Live Dashboard:** [Click here](https://wildfire-dashboard-xpkc.onrender.com/dashboard)



## Contents

- `01_data_preparation.ipynb` — data cleaning and aggregation

- `02_dashboard.ipynb` — visualization narrative and demonstration

- `dashboard.py` — standalone dashboard app (deployed via Render)

- `data/` — processed CSV files used by the dashboard

- `images/` — screenshots used in the narrative notebook
